STMs for scoring

${dataset}-asr.stm    : manual (disfluent) transcriptions of what the speaker said for use in ASR scoring.
${dataset}-fluent.stm : fluent manual transcription i.e. disfluencies removed from the complete manual transcript. Used in SGECF scoring.
${dataset}-gec.stm    : grammatical error corrected (GEC) transcription. Used in GEC WER and SGECF scoring.

N.B. GEC transcription is performed at a phrase level. Some phrases were excluded from this transcription. See the data description document for more details.
